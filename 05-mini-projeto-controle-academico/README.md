# Sistema de Controle Acadêmico Simples

Este mini-projeto foi desenvolvido como parte do meu portfólio de estudos em SQL.

O objetivo é simular um sistema acadêmico simples, com tabelas relacionadas para cursos, alunos, disciplinas, matrículas e notas.

## Objetivos do projeto

- Praticar criação de tabelas com SQL
- Trabalhar com chaves primárias e estrangeiras
- Inserir dados fictícios para testes
- Criar consultas com JOIN, GROUP BY, ORDER BY e funções de agregação
- Analisar dados acadêmicos de forma simples

## Estrutura do banco

O banco contém as seguintes tabelas:

- `cursos`
- `alunos`
- `disciplinas`
- `matriculas`
- `notas`

## Consultas desenvolvidas

- Listagem de alunos por curso
- Quantidade de alunos por curso
- Alunos matriculados por disciplina
- Disciplinas com mais matrículas
- Média final de cada aluno
- Situação de aprovação ou reprovação
- Ranking dos alunos por média geral

## Tecnologias utilizadas

- SQL
- SQLite
- Git e GitHub

## Aprendizados

Durante o desenvolvimento deste projeto, pratiquei modelagem de dados, relacionamento entre tabelas e consultas SQL para análise de informações acadêmicas.

Também organizei os scripts de criação, inserção e consulta para facilitar a leitura e execução do projeto.
